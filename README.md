# Think Tank — Bilingual Website (English + Arabic)

Powered by Hugo + Sveltia CMS + Netlify.

## Quick Setup

1. Upload all these files to a new GitHub repository named `thinktank-site`
2. In `static/admin/config.yml`, change `YOUR_GITHUB_USERNAME` to your GitHub username
3. In Netlify, click "Add new site" → "Import from GitHub" → select `thinktank-site` → Deploy
4. Visit `yoursite.netlify.app/admin/` to start editing content

## Your Team's Workflow

1. Go to `yoursite.netlify.app/admin/`
2. Log in with GitHub
3. Click a section (Research, News, Reports, Videos)
4. Click "New", fill in fields, switch to Arabic tab, fill in Arabic fields
5. Click Publish — live in 30 seconds
